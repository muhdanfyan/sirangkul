## Deskripsi Role

### 1. Administrator
- Manajemen pengguna (tambah, edit, hapus)
- Manajemen RKAM
- Pembayaran
- Audit Log
- Feedback

### 2. Pengusul
- Membuat proposal baru
- Melacak status proposal
- Upload dokumen pendukung (RAB/TOR)

### 3. Verifikator
- Memverifikasi kelengkapan proposal
- Menyetujui/menolak proposal ke tahap berikutnya

### 4. Kepala Madrasah
- Persetujuan akhir proposal
- Monitoring RKAM (read-only)
- Feedback

### 5. Bendahara
- Manajemen pembayaran
- Manajemen RKAM (edit)
- Pencatatan transaksi pencairan

### 6. Komite Madrasah
- Memberikan persetujuan/rekomendasi
- Pemantauan proposal
- Akses laporan
