# Skill: SiRangkul Design System & Token Standards

## Problem
Inconsistent color usage for budget sources makes the application confusing.

## UI Tokens
- **Dana BOS**: 
  - Text: Green-700
  - Background: Green-50
  - Border: Green-100
- **Dana Komite (KMT)**:
  - Text: Purple-700
  - Background: Purple-50
  - Border: Purple-100

## Consistency Rule
Any screen showing a budget source (Dashboard, Reporting, RKAM Table) MUST use these specific colors. Use `whitespace-nowrap` for currency tags to prevent splitting.
